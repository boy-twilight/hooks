# hooks

- 封装一组开发中常用的 hook
- hook 列表

---

```
useDrag<T>(container:string, dragData:Ref<T>)
```

| 参数      | 含义                  |
| --------- | --------------------- |
| container | 拖拽容器的类名或者 id |
| dragData  | 拖拽列表数据          |

| 返回值       | 含义                   |
| ------------ | ---------------------- |
| setDragStyle | 设置拖拽时候的元素样式 |
