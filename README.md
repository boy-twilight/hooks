# ms-draw：流程图绘制平台

1. 基于蚂蚁的 antv x6 实现的一个简易流程图绘画平台，实现了基本功能
2. 系统实现大概如下
   !['系统实现'](https://chart-twilight.gitee.io/image/%E6%B5%81%E7%A8%8B%E5%9B%BE%E7%BB%98%E7%94%BB.png)
3. 快捷键
   | 快捷键 | 操作 |
   | ---- | ---- |
   | crtl+c | 复制 |
   | crtl+x | 剪切 |
   | crtl+v | 粘贴 |
   | crtl+z | 撤销 |
   | ctrl+a | 全选 |
   | crtl+shift+z | 重做 |
   | backspace | 删除 |
   | crtl+鼠标滚轮 | 缩放画布 |
   | crtl+鼠标左键 | 拖动画布 |
